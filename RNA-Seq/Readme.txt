CuffDiff_Ecoli_Mono_vs_CoCulture.sh
Sample script for calculating differential expression. Samples in script are tophat2 output for E. coli mono- and co-cultures. 

Kallisto_Rpal.sh
Sample script for calculating RNA counts with Kallisto. Samples in script are R. palustris sequences. 

RNA_Assembly_Ecoli.sh
Script for mapping E. coli RNA sequences to the MG1655 Reference genome with tophat2. 

RNA_Assembly_Rpal.sh
Script for mapping R. palustris RNA sequences to the CGA009 Reference genome with tophat2.
